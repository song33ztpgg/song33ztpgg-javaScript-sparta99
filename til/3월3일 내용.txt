. Node.js는 프로그래밍 언어이다!?
Node.js는 JavaScript를 브라우저 없이 단독으로 실행할 수 있는 하나의 플랫폼입니다.  

npm은 라이브러리를 쉽게 사용하기 위한 개발된 패키지 관리자 형식의 프로그램 

express는 쉽게 서버프로그램을 구성할수 있게 만들어진 오픈 라이브러리 



http는 데이터를 주고 받을 때 사용되는 통신 규약 
웹브라우저 프로토콜등 지원하여 다른 웹서버에 데이터를 보낼수 있다 

쿠키란 : 웹브라우져 구현된 기술중 하나 , 상태를 저장하기 위해 사용  
브라우저는 가지고 있는 쿠기가 있다면 서버에 request할때 항상 가지고 있는 쿠키 데이터를 포함해서 보낸다 
쿠키는 암화화 없이 데이터를 주고 받기 때문에 클라이언트에서 마음데로 조작하기 쉬워보인다

세션은   웹 브라우저에 구현된 기술중 하나. 그러나 세션은 쿠키의 특성을 이용한 기술입니다.
 
서버프로그램이란 클라이언트에게 요청받아 응답 해주는 프로그램 

서버 컴퓨터란 서버 프로그램을 실행하고 있는 컴퓨터 

정적파일 : 변하지 않는파일

라우터를 이용해  method와 경로 http 요청받아 처리

REST API : URL<Headers,method등 네트워크 표현 수단을 이해하기 수윈표현으로 정의 
REST(Representational State Transfer)는 월드 와이드 웹과 같은 분산 하이퍼미디어 시스템을 위한 소프트웨어 아키텍처의 한 형식이다.
 

Validation : 검증한다 

/////////////// 
jwt : json 형태의 데이터로 안전하게  교환하게 사용해주는것 
데이터를 표현한 형식이다,서버에 저장하지 않아 데이터는 무상태이다 

무상태 : 서버가 죽다 살아다도 같은 동작을 한다 
상태보존 : 서버가 다시 부활할때 조금이라도 다른 동작을 한다
 
header.playload.signature 3가지 데이터를 포함하고 있다  

header : 
ㄴ alg :어떤알고리즘 이용해서 jwt만들었는지 확인용 
ㄴ typ : jwt 타입을 표현 

playload : 개발자가 원하는 데이터를 저장

signature : 토큰이 변조되지 않은 정상 토큰있는지 확인 ,
비밀키를 입력해서 인증 받는부분
